**NETI** is what happened when I needed to revise for a Java network-programming lab exam and decided building the study tool would teach me more than reading the notes would. It's an interactive trainer that drills the material instead of just displaying it.

## What it does

- **Quizzes** and **code-completion** exercises over Java networking concepts (sockets, streams, the lab syllabus).
- **Speed rounds** for timed recall practice.
- **Cloud sync** via Firebase, so progress follows you across devices.

## Built to ship fast

It's a single-file HTML app — auth, state, and UI in one place — deployed on GitHub Pages with no build step. The point wasn't architecture; it was getting a genuinely useful study tool live quickly, and learning the material by building the questions. Sometimes the fastest way to understand something is to make the thing that tests you on it.
